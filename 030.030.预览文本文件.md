#### 获取预览文件地址

POST ```/v1/preview/pdf```

* 请求地址 /v1/preview/text ```POST```
* 请求方式 JSON/Http POST
* 请求参数

| Properties     |  Type  | Description                                                |
|----------------|--------|------------------------------------------------------------|
|  *path         | string | path of the upload file.                                   |
|   uuid         | string | file id                                                    |

将返回预览文件的URL
```json
{"data":{"url":"xxx"}}
```
